215 /*MCSM0
216 Main Radio Control State Machine
217 Bit Field Name Description
218 7:6 Reserved
219 5:4 FS_AUTOCAL[1:0] Automatically calibrate when going to RX or TX, or back to IDLE
220  Setting When to perform automatic calibration
221  0 (00) Never (manually calibrate using SCAL strobe)
222  1 (01) When going from IDLE to RX or TX (or FSTXON)
223  2 (10) When going from RX or TX back to IDLE
224  automatically
225  3 (11) Every 4th time when going from RX or TX to IDLE
226  automatically
227  In some automatic wake-on-radio (WOR) applications, using
228  setting 3 (11) can significantly reduce current consumption.
229 3:2 PO_TIMEOUT Programs the number of times the six-bit ripple counter must
230  expire after XOSC has stabilized before CHP_RDYn goes low.
231  If XOSC is on (stable) during power-down, PO_TIMEOUT
232  should be set so that the regulated digital supply voltage has
233  time to stabilize before CHP_RDYn goes low (PO_TIMEOUT=2
234  recommended). Typical start-up time for the voltage regulator is
235  50 us.
236  If XOSC is off during power-down and the regulated digital
237  supply voltage has sufficient time to stabilize while waiting for
238  the crystal to be stable, PO_TIMEOUT can be set to 0. For
239  robust operation it is recommended to use PO_TIMEOUT=2.
240  Setting Expire count Timeout after XOSC start
241  0 (00) 1 Approx. 2.3 – 2.4 µs
242  1 (01) 16 Approx. 37 – 39 µs
243  2 (10) 64 Approx. 149 – 155 µs
244  3 (11) 256 Approx. 597 – 620 µs
245  Exact timeout depends on crystal frequency.
246 1 PIN_CTRL_EN Enables the pin radio control option
247 0 XOSC_FORCE_ON Force the XOSC to stay on in the SLEEP state.
248
