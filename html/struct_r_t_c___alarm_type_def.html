uint8_t RTC_AlarmDateWeekDay 
 Specifies the RTC Alarm Date/WeekDay.

If the Alarm Date is selected, this parameter must be set to a value in the 1-31 range. If the Alarm WeekDay is selected, this parameter can be a value of RTC_WeekDay_Definitions
